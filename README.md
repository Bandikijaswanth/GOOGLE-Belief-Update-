# GOOGLE-Belief-Update-
AIM

To compute the updated belief using a basic Bayesian update rule.

PROCEDURE
Initialize prior belief and observation likelihood
Apply the Bayesian product rule: belief = prior × observation
Substitute the given values
Compute the updated belief
Display the result
CODE
# Input values
prior = 0.7
observation = 0.6

# Belief update
updated_belief = prior * observation

# Output
print("Updated Belief:", updated_belief)
OUTPUT

Updated Belief: 0.42

RESULT

The updated belief is 0.42, showing how new observations refine prior knowledge in probabilistic systems.
